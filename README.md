# Bizaar eShop

https://bizaar-eshop.onrender.com/

## Description
An eCommerce shop built using the MERN stack along with Redux Toolkit, based largely on [Brad Traversy's eCommerce App](https://github.com/bradtraversy/proshop-v2).

## Technology Used
- MongoDB
- Express.js
- React
- Node.js
- Redux Toolkit
- Bootstrap
